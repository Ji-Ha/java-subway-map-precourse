## 🚀 기능 구현 목록

 - 코드를 구현하다 꼬일대로 꼬여서, 다시 처음부터 계획하고 시작한다.
 - 생각보다 매우 복잡하고, 복잡하다.
 
 - 출력
    1. 역 관리 화면을 출력하는 기능
    2. 노선 관리 화면을 출력하는 기능
    3. 구간 관리 화면을 출력하는 기능
    4. 메인 화면을 출력하는 기능
    5. 기능 선택 질문을 출력하는 기능
    6. INFO에 대한 내용 출력하는 기능
    7. 지하철 노선도를 출력하는 기능
    
 - 입력
    1. 숫자를 입력받는 기능
    2. 문자를 입력받는 기능
    
 - 데이터 관리
    1. Station을 추가하는 기능
    2. Station을 삭제하는 기능
    3. Line을 추가하는 기능
    4. Line을 삭제하는 기능
    5. 추가된 Line에 Station을 추가하는 기능
    6. 추가된 Line에 Station을 삭제하는 기능
    7. Line에 추가된 Station을 기록하는 기능
    8. Line에서 삭제된 Staion을 기록하는 기능 
    
 - 예외처리
    1. 숫자입력인데 다른 걸 입력할 때 예외처리
    2. 문자 입력인데 다른 것을 입력할 때 예외처리
    3. 빈 값 입력시에 예외처리
